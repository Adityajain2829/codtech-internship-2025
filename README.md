# Python Cybersecurity Toolkit

A collection of Python-based cybersecurity and networking tools developed during my internship at Codtech in 2025.

This repository contains beginner-to-intermediate security projects focused on:
- Cryptography
- File Integrity Monitoring
- Port Scanning
- Web Vulnerability Detection
- Networking Fundamentals

---

# Projects Included

## 1. AES-256 File Encryption Tool
A GUI-based file encryption and decryption tool using AES-256 encryption.

### Features
- AES-256 encryption/decryption
- Password-based key generation
- Tkinter GUI
- Secure file handling

---

## 2. File Integrity Monitor
Monitors files and detects:
- Added files
- Removed files
- Modified files

Uses SHA-256 hashing for integrity verification.

---

## 3. Python Port Scanner
A basic TCP port scanner that identifies open ports on a target system.

### Features
- Fast socket scanning
- Multiple common ports
- Timeout handling

---

## 4. Web Security Header Scanner
A basic web vulnerability detection tool for educational purposes.

### Features
- Detects security-related HTTP headers
- HTTPS verification
- Simple scanning automation

# Installation

Clone the repository:

```bash
git clone https://github.com/Adityajain2829/codtech-internship-2025.git
cd codtech-internship-2025
```

Install dependencies:

```bash
pip install -r requirements.txt
```

# Usage

Run any Python file:

```bash
python aes_encryptor.py
```

```bash
python port_scanner.py
```

```bash
python file_integrity_monitor.py
```
